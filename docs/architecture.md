# 🏗️ Krishiai Architecture

- **Frontend**: Simple HTML/CSS/JS (or React later) for UI.
- **Backend**: Flask (Python) for APIs and AI integration.
- **AI Models**: Pre-trained models for crop disease detection.
- **Data**: Sample datasets for testing/trials.

## Basic Flow
1) User uploads crop image on the frontend.
2) Frontend sends it to the backend API.
3) Backend runs the AI model and returns results.
4) Frontend shows the diagnosis and advice.
